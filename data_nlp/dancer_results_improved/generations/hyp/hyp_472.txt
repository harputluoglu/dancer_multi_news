– Google has issued a warning about a "serious" security flaw affecting more than half of the world's web browsers—and it's not all bad news "The vulnerability may corrupt memory in a way that could allow an attacker to execute arbitrary code in the context of the current user within Internet Explorer," the company says in a blog post "The phrase 'arbitrary code' means pretty much any software that the attacker chooses to run." Google says it's seen only "limited attacks" exploiting the vulnerability, which affects every version of the browser from version 6 through 11
